Program Name: MobyDick\
\
This program will search through the text of Moby Dick (or any text) and find a scrambled anagram, display the unscrambled anagram as found in the text, and display the line the unscrambled anagram is located in the text.\
\
To run the program:\
Step 1: Make sure you are in the directory above MobyDick.cpp\
	Ex. cd ~/cs1730/1730_program3\
Step 2: Compile the program by typing "make all"\
Step 3: Run the program by typing ./MobyDick.out " followed by the scrambled anagram you wish to unscramble based on the text followed by " < " and the name of the Moby Dick text file.\
	Ex. type: "./MobyDick.out taffomeatscenicicalinati < moby-dic.txt\
\
Ex. The program should display:\
	Line count is: 24967\
	Output is: scientific anatomical feat\
\
Note to the TA: This program completes in under 4 seconds on my Macbook.\
